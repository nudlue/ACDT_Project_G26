Following instructions

0. Move model file and img2 file to the raspberry pi(by using WinSCP)
1. Turn on raspbot(by using VNC viewer)
2. Turn on the terminal(black icon on the)
3. Type nano [file name].py
4. Copy and paste the codes in this github.(Do not copy once but copy with several times and use Ctrl+Shift+V when paste)
5. Open other termial.
6. Type python3 [file name].py
7. Then it will work
